# Global Tech challenge - User API

API in NodeJS using Nest Js. To make register and login users.



## Run

Using node

```bash
npm i && npm run start
```

using docker

```bash
docker image build -t user_api:1.0.0 .
docker container run -d -p 3333:3333 user_api:1.0.0
```
